File

public struct File : Saveable, Fetchable

A File object representes a file of binary data stored on the Parse server. This can be a image, video, or anything else that an application needs to reference in a non-relational way.

  • name

    The name of the file. Before the file is saved, this is the filename given by the user. After the file is saved, that name gets prefixed with a unique identifier.
