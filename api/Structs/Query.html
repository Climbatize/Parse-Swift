  • limit(_:)

    A limit on the number of objects to return. The default limit is 100, with a maximum of 1000 results being returned at a time.

    Note

    If you are calling findObjects with limit = 1, you may find it easier to use first instead.

    Declaration

    Swift

    public func first(options: API.Options = [], callbackQueue: DispatchQueue,
                  completion: @escaping (Result<ResultType, ParseError>) -> Void)

    Parameters

    options

    A set of options used to save objects.
    callbackQueue

    The queue to return to after completion. Default value of .main.
    completion

    The block to execute. It should have the following argument signature: ^(ParseObject *object, ParseError *error). result will be nil if error is set OR no object was found matching the query. error will be nil if result is set OR if the query succeeded, but found no results.
