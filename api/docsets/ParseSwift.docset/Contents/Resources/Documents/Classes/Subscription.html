Subscription

@available(macOS 10.15, iOS 13.0, watchOS 6.0, tvOS 13.0, *)
open class Subscription<T> : ParseSubscription, ObservableObject where T : ParseObject

A default implementation of the ParseSubscription protocol. Suitable for ObjectObserved as the subscription can be used as a SwiftUI publisher. Meaning it can serve indepedently as a ViewModel in MVVM.
